# Contributing to Pr0mt_X Text

Thanks for helping improve Pr0mt_X Text.

## Before You Start

- Search existing issues before opening a new one.
- Keep changes focused and consistent with the existing native Win32 design.
- Do not commit generated build files or local `*.dat` files.
- Contributions must be compatible with the PolyForm Noncommercial License 1.0.0.

## Building

Install CMake 3.16 or newer and Visual Studio Build Tools with the Desktop development with C++ workload, then run:

```bat
build.bat
```

The executable is written to `build\Release\Pr0mt_X Text.exe`.

## Pull Requests

1. Build the Release configuration and confirm it completes without warnings.
2. Test the affected workflow on Windows 10 or Windows 11.
3. Describe the behavior change and the manual checks you performed.
4. Include screenshots for visible UI changes.

By submitting a contribution, you agree that it may be distributed under the repository's license.
